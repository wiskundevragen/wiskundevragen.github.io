Overzicht veelvoorkomende functies met hun primitieve/integraal

Functie Primitieve Opmerking
f(x)=x^n F(x)=1/(n+1) * x^(n+1) + C voor x ongelijk aan -1
f(x)=a F(x)=ax + C
f(x)=1/x F(x)=ln(x) + C
f(x)=ln(x) F(x) = xln(x) - x + C
f(x)=e^x F(x)=e^x + C
f(x)=a^x F(x)=a^x/ln(a) + C
f(x)=g(x)+h(x) F(x)=∫(g(x)+h(x)) + C Somregel
f(x)=g'(x)h(x) F(x)=g(x)h(x) - ∫g(x)h'(x) + C "Omgekeerde Productregel"; partieel integreren (geen examenstof)
f(x)=sin(x) F(x)=-cos(x) + C
f(x)=cos(x) F(x)=sin(x)
f(x)=arcsin(x) F(x)=x*arcsin(x) + sqrt(1 - x^2) + C Uitleg
f(x)=arccos(x) F(x)=x*arccos(x) - sqrt(1-x^2) + C Uitleg
f(x)=arctan(x) F(x)=x*arctan(x) - 0.5ln(1+x^2) Uitleg
